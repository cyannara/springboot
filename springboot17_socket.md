## STORM
- Simple(or Streaming) Text Oriented Messaging Protocol
- 텍스트 기반 프로토콜
- STOMP 클라이언트가 프로토콜을 지원하는 모든 메시지 브로커와 통신할 수 있도록 상호 운용 가능한 와이어 형식을 제공합니다 
- STOMP는 하위 수준의 WebSocket 위에 작동하는 하위 프로토콜

### 메시지 브로커(메시지 지향 미들웨어 제품)
- Spring Framework
- `RabbitMQ`
- Apache ActiveMQ
- Fuse Message Broker
- HornetQ
